# ADR-005: Temperature Sensor Simulation

## Status

Accepted

---

## Context

ColdGuard requires temperature monitoring functionality.

Real IoT hardware is outside the scope of the capstone project.

---

## Decision

Temperature readings will be simulated.

---

## Rationale

Simulation allows:

- Demonstration of monitoring workflows
- Alert generation testing
- Risk analysis validation
- Capstone completion without hardware dependencies

---

## Alternatives Considered

### Real IoT Sensors

Pros:
- Real-world integration

Cons:
- Additional hardware cost
- Increased complexity
- Project scope expansion

---

## Decision Outcome

Sensor simulation provides sufficient functionality for demonstrating monitoring capabilities.
