# ADR-003: Deployment Platform Selection

## Status

Accepted

---

## Context

ColdGuard requires cloud hosting for:

- Demonstration
- Testing
- Continuous deployment
- Public accessibility

---

## Decision

Render will be used as the deployment platform.

---

## Rationale

Render provides:

- Simple deployment workflows
- GitHub integration
- Automatic deployments
- Health checks
- Environment variable management
- Free and low-cost hosting options

---

## Alternatives Considered

### AWS

Pros:
- Enterprise-grade scalability

Cons:
- Increased complexity
- Higher learning curve

### Azure

Pros:
- Enterprise ecosystem

Cons:
- Higher administrative overhead

---

## Decision Outcome

Render offers the best balance between simplicity, functionality, and educational value.
